# ROG_GRUB_Theme
GRUB Theme for ROG Fans

![preview.png](preview.png)

## Getting Started

Just Copy the Following Command in Your Terminal 

```bash
git clone https://github.com/thekarananand/ROG_GRUB_Theme
cd ROG_GRUB_Theme
chmod +x install.sh
sudo ./install.sh
```



*the theme is based on an image on the internet.
